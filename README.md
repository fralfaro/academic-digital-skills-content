## ¿Qué es este repositorio?

Este es el repositorio de entregables del curso. Aquí encontrarás las instrucciones de cada actividad y podrás ver el trabajo de tus compañeros.

## Estructura

```
academic-digital-skills-entregables/
├── trabajos/
│   ├── 01-github/     # Módulo 1 — Repositorio y README
│   ├── 02-colab/      # Módulo 2 — Notebook con análisis
│   ├── 03-overleaf/   # Módulo 3 — Informe PDF
│   └── 04-quarto/     # Módulo 4 — Presentación
└── README.md
```

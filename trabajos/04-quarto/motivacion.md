# 🎬 Motivación — Quarto

## ¿Por qué aprender Quarto?

Imagina hacer una presentación donde los gráficos se actualizan solos cuando cambian los datos. Eso es Quarto. En Ingeniería vas a presentar resultados todo el tiempo, y muchas veces esos resultados vienen de código y datos que siguen cambiando. Quarto te deja crear presentaciones, reportes y hasta sitios web que combinan texto, código y visualizaciones en un solo archivo. En vez de copiar y pegar gráficos a mano cada vez que algo cambia, dejas que tu documento se reconstruya solo. Es el siguiente nivel para mostrar tu trabajo.

## El problema que resuelve

Hacer una presentación bonita en PowerPoint toma tiempo, y si además incluye código, tablas o gráficos de tus análisis, mantenerlos actualizados se vuelve imposible. Cambias un dato y tienes que rehacer manualmente cada gráfico, cada número, cada lámina. Tarde o temprano algo queda desactualizado y presentas resultados que ya no calzan con tus datos. Es trabajo repetitivo, frágil y fácil de equivocar.

## Lo que vas a poder hacer

- Crear presentaciones y reportes reproducibles que se regeneran solos cuando cambian los datos.
- Integrar código de Python o R directamente en el documento, con sus resultados incluidos.
- Publicar tu trabajo en la web de forma automática, listo para compartir con un link.
- Reutilizar un mismo proyecto para informes, presentaciones y tu portafolio personal.

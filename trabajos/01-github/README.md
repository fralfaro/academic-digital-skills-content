# Módulo 1 · GitHub

## 🎯 Objetivo

Clonar el repositorio del curso, personalizar tu perfil de GitHub con una foto y una descripción breve.

---

## 📌 Pasos a seguir

### 1. Clona el repositorio plantilla

Ve a:

```
https://github.com/fralfaro/academic-digital-skills-entregables
```

Haz clic en **Use this template** → **Create a new repository** y ponle el nombre que quieras.

---

### 2. Agrega una foto a tu perfil

Ve a tu perfil de GitHub → **Edit profile** → sube una foto tuya o un avatar.

> 💡 Puedes generar un avatar en [https://avatars.dicebear.com](https://avatars.dicebear.com) o usar cualquier imagen cuadrada.

---

### 3. Completa tu perfil

Agrega al menos:

- **Bio**: una línea corta sobre ti (ej: *"Estudiante de Ingeniería Civil · UTFSM"*)
- **Ubicación**: tu ciudad
- **Link** (opcional): tu LinkedIn, Instagram, etc.

Así debería verse tu perfil al terminar:

![Ejemplo de perfil](../../img/profile_example.png)

---

## ✅ Checklist antes de publicar en el foro

- [ ] Repositorio creado desde la plantilla y visible como **público**
- [ ] Foto de perfil subida
- [ ] Bio con al menos una línea de descripción
- [ ] Link o red social agregada (opcional)

---

## 🔗 Entrega en el foro

Publica el link de tu **perfil de GitHub** en el foro del módulo. Ejemplo:

```
https://github.com/tu-usuario
```

Luego visita el perfil de un compañero y comenta: ¿se ve completo? ¿qué le agregarías?

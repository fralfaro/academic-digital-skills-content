# Módulo 3 · Overleaf / LaTeX

## 🎯 Objetivo

Escribir un documento matemático corto en Overleaf, exportarlo como PDF y subirlo a tu repositorio de GitHub.

---

## 📌 Pasos a seguir

### 1. Abre la plantilla en Overleaf

👉 [**Abrir plantilla en Overleaf**](https://www.overleaf.com/latex/templates/proof-portfolio-template/vzzvphktcgdr)

Haz clic en **Open as Template** para crear tu propia copia editable.

---

### 2. Elige un tema

Escoge el que más te llame la atención:

| # | Tema |
|---|------|
| A | Ecuaciones cuadráticas |
| B | Teorema de Pitágoras |
| C | Media y varianza |
| D | Números primos |
| E | Probabilidad básica |
| F | El número $e$ o $\pi$ |
| G | Otro tema matemático que te guste |

---

### 3. Pídele el contenido a la IA

Copia este prompt en Claude, ChatGPT o el asistente que prefieras, reemplazando `[TEMA]` por el que elegiste:

> *"Quiero escribir un documento corto en LaTeX sobre **[TEMA]** usando esta plantilla: https://www.overleaf.com/latex/templates/proof-portfolio-template/vzzvphktcgdr*
>
> *Necesito el código LaTeX con: título, autor, fecha, una definición (`\begin{defn}`) y una proposición o teorema (`\begin{prop}` o `\begin{theorem}`) con su demostración (`\begin{proof}`). Que sea simple y breve."*

La IA te entregará el código LaTeX completo listo para pegar.

---

### 4. Pega, compila y descarga

1. Copia el código que te dio la IA en Overleaf
2. Haz clic en **Recompilar** (`Ctrl+Enter`) — si hay errores, pídele a la IA que los corrija
3. Descarga con **Menú → Descargar PDF**
4. Renombra el archivo como `informe.pdf` y súbelo a tu repositorio de GitHub

---

## ✅ Checklist antes de publicar en el foro

- [ ] El documento tiene título, autor y fecha con tu nombre
- [ ] Incluye al menos una definición y una proposición o teorema
- [ ] Compila sin errores (sin texto en rojo)
- [ ] El archivo `informe.pdf` está en tu repositorio de GitHub

---

## 🔗 Entrega en el foro

Publica el link al PDF en tu repositorio. Ejemplo:

```
https://github.com/tu-usuario/academic-digital-skills-entregables/tree/main/trabajos/03-overleaf/informe.pdf
```

Luego revisa el informe de un compañero y comenta: ¿qué tema eligió? ¿se ve bien?
